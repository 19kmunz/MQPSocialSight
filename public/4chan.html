Design Overview

4chan is a bulletin-board forum social media, where users can post an image or video. This creates a thread, which can be replied to. Different boards are dedicated to different topics, but all are image-based. All user posts are anonymous unless you fill in the [Name] category on posts, but other users can input the same name as you. 4chan has two versions, with 4channel.org being safe for work content and 4chan.org being all content.

Topic-Based Boards

4chan features boards categorized per topic. On the safe for work site, these have five major categories: Japanese Culture, Video Games, Interests, Creative and Other. Post an image to start a thread, which can then be replied to by other users. Each board has different moderation and posting rules.

Anonymous Posting

4chan has no account or login, only an optional [Name] tag or a more secure "tripcode" to validate identity. Unless a user posts their contact information, there is no way to message privately.

Extensive User Settings

To counter 4chan's free speech policies, it allows users to establish their own precise settings, including word filters, image blur, video autoplay and custom styling. These settings apply per board, but can be exported for future use.
